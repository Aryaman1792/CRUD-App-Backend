# CRUD App Backend (OOP Architecture)

A production-ready **CRUD backend application** built using **Node.js, Express, TypeScript, and MongoDB**, following clean **Object-Oriented Programming (OOP)** principles.

---

## Features

- Full CRUD operations
- Clean OOP architecture (Controller → Service → Repository)
- MongoDB with Mongoose
- Search, filter, pagination
- Centralized error handling
- TypeScript support
- Production & development scripts
